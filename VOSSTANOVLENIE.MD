По этой прошивке Bluetooth-функции вынесены в один системный аккорд: **все четыре нижние основные клавиши одновременно**. В `chords.dtsi` это аккорд `BLP BLR BLM BLI` для левой стороны и `BRP BRR BRM BRI` для правой; без тамба он даёт `BT_SEL 0`, с внутренним тамбом — `BT_SEL 1`, с внешним тамбом — `BT_SEL 2`, с двумя тамбами — `BT_CLR`. ([GitHub][1])

Для ориентации по кнопкам:

* **внутренний тамб** — это тамб с `Space`;
* **внешний тамб** — это тамб с `Backspace`.
  Это прямо следует из keymap: на левой половине `LOT = BSPC`, `LIT = SPACE`, на правой половине `RIT = SPACE`, `ROT = BSPC`. ([GitHub][1])

В практической таблице этого же репозитория зафиксировано:

* `низ1 + низ2 + низ3 + низ4` → `BT_SEL 0`;
* тот же аккорд + **правый тамб** → `BT_SEL 1`;
* тот же аккорд + **левый тамб** → `BT_SEL 2`;
* тот же аккорд + **два тамба** → `BT_CLR`.
  При этом в пояснении к таблице указано, что **левый тамб = внешний**, а **правый тамб = внутренний**. ([GitHub][2])

Точная последовательность для возврата нужного Bluetooth-слота:

1. Нажать **все 4 нижние основные клавиши одновременно**.
   Это переключит клавиатуру на **Bluetooth-профиль 0**. ([GitHub][2])

2. Если нужный хост не на профиле 0, нажать:

   * **все 4 нижние основные клавиши + внутренний тамб (Space)** → профиль 1;
   * **все 4 нижние основные клавиши + внешний тамб (Backspace)** → профиль 2. ([GitHub][2])

Точная последовательность для очистки bond на самой клавиатуре:

1. Нажать **все 4 нижние основные клавиши одновременно**.
2. Одновременно с ними нажать **оба тамба** — и `Space`, и `Backspace`.
   Это выполняет `BT_CLR`, то есть **очистку текущего Bluetooth-профиля**. ([GitHub][2])

Практический порядок восстановления связи без графического интерфейса:

1. Сначала попробовать профиль 0:
   **низ1 + низ2 + низ3 + низ4**
2. Затем профиль 1:
   **низ1 + низ2 + низ3 + низ4 + Space**
3. Затем профиль 2:
   **низ1 + низ2 + низ3 + низ4 + Backspace**
4. Если ни один профиль не помогает, очистить текущий профиль:
   **низ1 + низ2 + низ3 + низ4 + Space + Backspace**
5. После очистки снова выбрать нужный слот, обычно сначала профиль 0, и выполнить новое сопряжение с хостом через `bluetoothctl`. ([GitHub][2])

Если нужно совсем буквально по раскладке, то этот системный аккорд — это **все четыре клавиши нижнего ряда основной матрицы** на одной стороне устройства.```
